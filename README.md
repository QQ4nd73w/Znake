# Zмейка

Классическая игра «Змейка» с современными улучшениями. В проекте есть анимированное меню, система баффов и локальные рекорды. Игра выполнена на `HTML5 Canvas`, а логика реализована на чистом `JavaScript` с сохранением настроек в `localStorage`.

Стек
-----
- Frontend: HTML5, CSS3, JavaScript
- Backend: нет
- DB: нет, хранение данных в `localStorage`

Как запустить локально
----------------------
1. Клонируйте репозиторий и перейдите в папку проекта.
2. В папке проекта запустите файл start.bat(в зависимости от вашего браузера), проект откроется в режиме инкогнито.
